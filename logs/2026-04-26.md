## 🧪 Issues Encountered

---

## 📚 Key Learnings
- GitHub file creation is path-based, not folder-based
- Creating `folder/file.md` inside a folder already places it within that folder again if misused
- Homelab documentation benefits from strict structural discipline
- Treating system components as “modules” improves clarity and scalability
